Comando para listar carpetas y archivos: ls


La forma de escribir el comando en consola es de la siguiente forma: ~$ ls ¿Que hace la opcion -l del comando ls? Al momento de escribir: ~$ ls -l nos listara los archivos en forma de lista con sus permisos, usuario, grupo, tamaño, fecha de creación y nombre. ¿Que hace la opcion -a del comando ls? Al momento de escribir: ~$ ls -a nos permite visualizar los archivos ocultos.

Comando para crear un archivo: touch

La forma de escribir el comando en consola es de la siguiente forma: ~$ touch [Nombre del archivo]

Comando para crear un directorio: mkdir

La forma de escribir el comando en consola es de la siguiente forma: ~$ mkdir [Nombre del directorio]

Comando para eliminar un archivo: rm

La forma de escribir el comando en consola es de la siguiente forma: ~$ rm [Nombre del archivo que se desee eliminar] Opcion para eliminar un directorio: rmdir La forma de escribir el comando en consola es de la siguiente forma: ~$ rmdir [Nombre de la carpeta a eliminar]

Comando Nano

El comando nano es un editor de archivos simple, el comando para ingresar es:
~$ nano [Nombre del archivo que se desee editar]

Comando para guardar cambios: ctrl+o para guardar los cambios realizados.

Comando para salir del editor de texto: ctrl+x para salir del editor de archivos.
